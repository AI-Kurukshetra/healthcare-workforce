# Admin Role

## Overview

The **Admin role** has full control of the Healthcare Workforce Management Platform.

Admins manage **users, departments, system settings, and organization-wide analytics**.

---

# Admin Dashboard

The Admin dashboard shows **system-wide workforce metrics**.

### Widgets

* Total Staff
* Department Overview
* Credential Alerts
* Open Shifts
* System Notifications
* Workforce Utilization

---

# Staff Management

Admins manage the entire workforce.

### Features

* Create staff accounts
* Update staff profiles
* Assign roles
* Deactivate staff
* Manage employment types

---

# Department Management

Admins manage hospital departments.

### Features

* Create departments
* Create units
* Assign staff to departments
* Update department structure

---

# Scheduling Management

Admins can control the entire scheduling system.

### Features

* Create shifts
* Assign staff
* Modify schedules
* Monitor open shifts

---

# Credential Compliance

Admins manage certification policies.

### Features

* Define credential requirements
* Monitor expiring credentials
* Generate compliance alerts

---

# Reporting & Analytics

Admins have access to full analytics.

### Reports

* Workforce utilization
* Overtime costs
* Shift coverage
* Staff productivity
* Compliance reports

---

# Notification Management

Admins can manage system notifications.

### Examples

* Policy updates
* Compliance alerts
* Workforce announcements

---

# System Configuration

Admins manage system settings.

### Features

* User role management
* Organization settings
* Integration configuration
* Security settings
